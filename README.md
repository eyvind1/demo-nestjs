# test-backend-nest

## Installation

Install the latest version of node js

## Installation nest js
```bash
$ npm install @nestjs/cli -g
```

## Clone the project and install the modules in the same directory
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
